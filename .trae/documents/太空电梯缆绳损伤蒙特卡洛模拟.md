# 太空电梯缆绳损伤蒙特卡洛模拟实现计划（50年时间尺度）

## 1. 问题分析
- **数据来源**：`leo_debris_environment_esa.csv` 包含不同大小碎片的空间密度和相对速度
- **物理模型**：使用用户提供的损伤公式 `r + r*sin(Φ)/tan(θ) ≥ w*f/2`
- **关键参数**：
  - 缆绳宽度：10 cm
  - 损伤阈值：50% 宽度
  - 停运时间：7天/次断裂
  - 撞击过程：泊松过程
  - 入射方向：各向同性
  - 模拟时间：50年
  - 太空垃圾增长：撞击率随时间逐步提升

## 2. 实现步骤

### 2.1 数据处理
- 读取并解析 CSV 数据
- 计算初始撞击率
- 设计太空垃圾增长模型（线性或指数增长）

### 2.2 蒙特卡洛模拟核心
- **时间步进**：模拟50年的时间，按天计算
- **撞击率动态调整**：根据年份逐步提升撞击率
- **撞击事件生成**：基于泊松过程生成撞击事件
- **损伤计算**：
  - 随机生成碎片大小、入射角度
  - 使用物理模型计算每次撞击的损伤宽度
  - 累计损伤宽度
- **断裂判断**：当累计损伤≥5 cm时，触发断裂
- **停运时间计算**：记录每次断裂并累计停运天数

### 2.3 可视化实现
- **损伤分布**：绘制缆绳的受损图，显示不同位置的损伤
- **时间序列**：绘制50年中损伤累计和停运情况
- **趋势分析**：显示随时间增长的撞击率和损伤情况
- **统计分析**：生成50年损伤统计报告

### 2.4 代码结构
- `main.py`：主模拟逻辑
- `physics.py`：物理模型计算
- `visualization.py`：可视化函数
- `utils.py`：工具函数
- `debris_model.py`：太空垃圾增长模型

## 3. 关键算法

### 3.1 撞击率计算与增长模型
- 初始撞击率：使用空间密度 (SPD) 和相对速度 (Vr) 计算
- 增长模型：可选择线性增长（每年固定百分比）或指数增长

### 3.2 泊松过程模拟
根据动态调整的撞击率生成符合泊松分布的撞击事件

### 3.3 损伤计算
实现用户提供的物理模型，考虑碎片大小和入射角度对损伤的影响

### 3.4 累计损伤管理
跟踪缆绳的累计损伤，判断是否达到断裂阈值

## 4. 输出结果
- 50年停运天数统计
- 年际损伤趋势分析
- 缆绳损伤分布可视化
- 撞击事件统计分析
- 不同大小碎片的贡献分析
- 太空垃圾增长对损伤的影响评估
